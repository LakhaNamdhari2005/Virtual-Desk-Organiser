# Virtual Desk Organiser

AI-powered smart desk organiser using ESP32-S3 and e-paper display.

## Features
- Weather updates
- Calendar sync
- Quotes
- To-do list
- Alarm reminders
- Web dashboard

## Hardware
- ESP32-S3 Pico
- DESPI-C02
- E-paper display
- PIR sensor
- Solar panel
